# Cinema Online Hub

**Tác giả:** Trần Quốc Việt Hùng  
**Đối tượng:** Người dùng (User)

### 🚀 Các chức năng hệ thống (6 chức năng)
```http
GET    /users         - Read All
GET    /users/:id     - Read One
GET    /users/me      - Read Me
PATCH  /users/me      - Update
PATCH  /users/:id/ban - Toggle Ban
DELETE /users/:id     - Delete
```
