# cinema-online-hub
Trần Quốc Việt Hùng: đối tượng user
có 6 chức năng :
    Read All (GET /users)
* Read One (GET /users/:id)
* Read Me (GET /users/me)
* Update (PATCH /users/me)
* Toggle Ban (PATCH /users/:id/ban)
* Delete (DELETE /users/:id)
